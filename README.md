Este repositório contém uma coleção de desafios de lógica de programação em **Java**, abrangendo desde sistemas de automação bancária até a implementação de algoritmos clássicos de ordenação de vetores.

---

## 🚀 O que há no repositório?

O projeto está estruturado em dois grandes pilares:

### 1. Sistema de Caixa Eletrônico 💵
Simulação completa de um terminal de autoatendimento, focando em:
* **Lógica de Saque:** Distribuição eficiente de cédulas.
* **Modularização:** Separação de responsabilidades (Input, Processamento e Output).
* **Validação:** Tratamento de entradas do usuário.

### 2. Laboratório de Algoritmos (Sorting) 🧪
Implementação dos principais métodos de ordenação para análise de complexidade e performance:
* **Quick Sort:** Otimizado com pivô e recursividade.
* **Merge Sort:** Estratégia de "Dividir para Conquistar".
* **Bubble Sort (Classic & Optimized):** Comparativo de métodos de troca.
